# 発表プロンプター

演台のiPhoneで使う、発表者用プロンプターWebアプリです。1ファイル完結・オフライン動作・通信不要。

**▶ アプリを開く: https://otanoritaka.github.io/presentation-prompter/**

## 機能

- **7分タイマー** — 時計をタップで開始/一時停止。6:00で黄色、6:40で赤点滅の警告
- **進行ナビ** — ページ送りごとに予定表と比較して「◯秒 先行 / ◯秒 遅れ」を表示
- **スライド別カウントダウン** — 各スライドの目安秒数と残り時間
- **文字サイズ調整** — A− / A＋（設定は端末に記憶）
- **原稿編集** — ✎ボタンでその場修正、端末に保存。「初期原稿に戻す」も可能
- **ページ送り** — ボタンまたは左右スワイプ。次スライドの予告表示つき
- **画面スリープ防止** — 対応端末では自動で画面を維持

## iPhoneでの使い方

1. SafariでこのページのURLを開く
2. 共有ボタン → **「ホーム画面に追加」**（全画面のアプリとして使えます）
3. 念のため 設定 → 画面表示と明るさ → **自動ロック → なし** に変更
4. 演台に置き、発表開始と同時に時計をタップ

## カスタマイズ

原稿・目安秒数は `index.html` 内の `DEFAULTS` 配列を書き換えるだけです。
`《…》` で囲んだ部分は強調（ピンク）、行頭 `◆` は演出指示（黄色）として表示されます。
